# Expense Tracker — Трекер расходов

**Автор:** Шайхутдинов Айзат
**Язык:** Python  
**Библиотеки:** tkinter, pandas  
**Формат данных:** JSON  
**Система контроля версий:** Git

## Описание программы

Expense Tracker — простое приложение для учёта личных расходов с графическим интерфейсом. Позволяет добавлять расходы с указанием суммы, категории и даты, фильтровать записи по категории и дате, а также подсчитывать общую сумму расходов. Все данные сохраняются в файл `expenses.json`.

## Как использовать

1. Установите Python и необходимые библиотеки:
   ```bash
   pip install pandas
   ```
2. Запустите приложение:
   ```bash
   python expense_tracker.py
   ```
3. Введите сумму (положительное число), категорию и дату (ГГГГ-ММ-ДД), затем нажмите «Добавить расход».
4. Используйте фильтры для поиска по категории или дате.
5. Сумма всех расходов отображается внизу окна.
6. Данные сохраняются автоматически при добавлении новой записи.
7. Для работы с Git:
   - Инициализируйте репозиторий: `git init`
   - Добавьте файлы: `git add .`
   - Сделайте коммит: `git commit -m "Initial commit"`
   - Создайте репозиторий на GitHub/GitLab и отправьте туда проект.
