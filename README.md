Дневник погоды
Автор: Шайхутдинов Айзат

Описание программы
Приложение для ведения дневника погоды: добавление, просмотр, фильтрация записей с сохранением в формате JSON.

Требования
Python 3.x
Запуск
Установите Python (если он не установлен).
Клонируйте репозиторий: git clone <ссылка>
Запустите: python weather_diary.py
Примеры использования
Добавьте запись с датой 01.01.2026, температурой 5, описанием "Солнечно", осадками "Нет".
Используйте фильтрацию по температуре > 10°C.
